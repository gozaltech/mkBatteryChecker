# mkBatteryChecker

A lightweight Windows system tray application that monitors the battery level of Apple Magic Keyboards connected to your PC.

## Features

- System tray icon with live battery percentage tooltip
- Low battery notifications at 25%, 20%, 15%, 10%, and 5%
- Automatic device detection — selects the first found device on startup
- Device switcher via right-click context menu
- Start with Windows toggle in tray menu

## Requirements

- An Apple Magic Keyboard (Bluetooth or USB) paired and connected
